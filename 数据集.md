# 数据集

+++

MMLU 是一个用来评估大语言模型“通识学科能力”的基准测试，名字全称是 **Massive Multitask Language Understanding**。

它的典型形式是：

- **覆盖很多学科**：包含 **57 个学科领域**（例如数学、物理、化学、生物、历史、法律、经济、计算机科学、医学等）。
- **题型以选择题为主**：给一段题干 + 若干选项，让模型选出正确答案。
- **主要指标**：**Accuracy（准确率）**，通常会报告总体平均分，以及不同学科/类别（STEM、人文、社科、其他）的分数。
- **评测设置**：常见有 **zero-shot / few-shot**（不给示例/给少量示例）两种，比较模型“直接答题”与“看过少量示例后答题”的能力差异。

大家用 MMLU 主要是想回答一个问题：
**模型是否真的懂得跨学科知识，并能在不同领域稳定作答？**

但它也有局限：

- 以**选择题**为主，和真实对话/开放式任务不完全一致
- 可能受到**训练数据污染**（模型见过类似题目）影响
- 对“过程性推理、工具使用、长链条推理”覆盖有限

如果你还看到类似名字：**CMMLU**（中文 MMLU）、**MMLU-Pro**（更难/更注重推理）等，它们基本都是在 MMLU 思路上做的扩展或改进。
